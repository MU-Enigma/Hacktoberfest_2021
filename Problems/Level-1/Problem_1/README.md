# Problem 1

Given a string as input, return the reversed string with the caps swapped.

## Input Format

```md
First line represents T which is the number of test cases.
First line of each test case represents the string, S.
```

### Sample Input

```c
2
eNiGMA
fLUmE
```

### Sample Output

```c
amgInE
eMulF
```
